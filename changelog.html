0.9.0 (Preview)

2025-01-15

New

  • Category‑aware ScreenScraper engine with caching and fallbacks
  • Support for 17 systems with automatic RetroArch core selection
  • Dedicated media server for covers, thumbnails, and videos
  • SwiftUI UI with adaptive grid, dark mode, and drag & drop

Improvements

  • 7‑day trial with Device ID in Keychain and offline grace period
  • Robust error handling for timeouts, rate limits, and retries

Known

  • No HTTP range resume for media downloads yet
  • SHA1 hashing is slow for very large disc images
