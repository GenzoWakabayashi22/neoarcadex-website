Changelog

Track the evolution of NeoArcadeX with our version history

Version 1.2.0

January 15, 2025

New Features

  • Added support for PlayStation 2 emulation via PCSX2
  • New grid view customization options with adjustable thumbnail sizes
  • Automatic controller configuration for popular USB gamepads
  • Enhanced artwork scraper with support for additional metadata sources

Improvements

  • Faster game library scanning and indexing
  • Improved memory usage for large collections (10,000+ games)
  • Better Apple Silicon optimization

Version 1.1.0

November 28, 2024

New Features

  • Added filter and search functionality with real-time results
  • Support for custom game categories and collections
  • Backup and restore functionality for game library

Bug Fixes

  • Fixed artwork download issues for certain game databases
  • Resolved crash when launching games with special characters in path
  • Fixed dark mode theme inconsistencies
  • Corrected ROM file detection for .cue/.bin format games

Version 1.0.0

September 10, 2024

Initial Release

  • Beautiful game library interface with grid and list views
  • Support for 20+ classic gaming platforms
  • Integration with popular emulators (RetroArch, MAME, Dolphin, etc.)
  • Automatic metadata and artwork download
  • Customizable themes and UI layouts
  • Game favorites and recently played tracking
  • Native macOS design with dark mode support
