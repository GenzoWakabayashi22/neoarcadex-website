What is NeoArcadeX

NeoArcadeX is a native macOS frontend that turns managing and launching retro games into a fast, elegant experience. It’s written in Swift using SwiftUI and Swift Concurrency (async/await, Actors), integrates RetroArch, and ships with a highly optimized ScreenScraper engine for instant artwork and metadata.

Key features

  • Smart trial: 7 days, secure Device ID in Keychain, 24h offline grace
  • Pro license: permanent unlock for €5, no invasive DRM
  • 17 systems with optimal RetroArch cores and automatic detection
  • Category‑aware ScreenScraper engine with caching and resilient retries
  • Dedicated media server for fast covers, thumbnails, and videos
  • Modern UI: adaptive grid, context menus, drag & drop, dark mode
  • Play stats: time played, last played, launch count; JSON export/import

Tech stack

  • SwiftUI + MVVM
  • Swift Concurrency (async/await), Actors for thread safety
  • FileSystemActor for console detection from paths
  • Multi‑level caching for trial, metadata, and media

Recommended folder structure

ROMs/
├── NeoGeoAES/       (.zip MAME)
├── NeoGeoCD/        (.cue/.bin/.chd)
├── Arcade/          (.zip)
├── Amiga/           (.adf, .lha)
├── PSX/             (.cue/.bin/.chd, .pbp)
├── Saturn/          (.cue/.bin/.chd)
├── SNES/            (.sfc, .smc, .zip)
├── Megadrive/       (.md, .bin, .gen)
├── ... (other systems)
├── Covers/          (per system)
├── Thumbs/          (thumbnails)
└── Videos/          (mp4)

Not (yet) included

  • No built‑in cloud sync
  • No support for newer consoles like GC/Wii/PS2/PSP/DS/N64
  • No per‑game RetroArch overrides via the UI
